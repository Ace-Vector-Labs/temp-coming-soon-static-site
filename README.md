## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Basic HTTP server (optional, for local development)

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Ace-Vector-Labs/temp-coming-soon-static-site.git
   cd temp-coming-soon-static-site
   ```

2. Open `index.html` in your web browser, or serve it locally:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```

3. Visit `http://localhost:8000` (or the port specified by your server)
